## 🚀 Proyectos Destacados

### 🏪 [RetailManager](https://github.com/crishof/RetailManager) — Sistema ERP Comercial
> **Demo en vivo:** [retail-manager-iota.vercel.app](https://retail-manager-iota.vercel.app)

Sistema de gestión comercial modular basado en **17+ microservicios** independientes.

**Stack:** Java 25 · Spring Boot 4 · Spring Cloud Gateway · Eureka · RabbitMQ · PostgreSQL · Docker · Angular 17 · Resilience4j · Feign Client

- ✅ API Gateway con enrutamiento y balanceo de carga
- ✅ Service Discovery con Eureka
- ✅ Mensajería asíncrona con RabbitMQ
- ✅ Base de datos por servicio (database-per-service pattern)
- ✅ Docker Compose para infraestructura local
- ✅ Integración con Cloudinary para imágenes
- ✅ Multi-almacén, multi-sucursal, tipos de cambio
